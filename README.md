Simple API to generate and verify 15-digit pin.

## To run on your local machine
- create virtual environment
- `pip3 install flask`
- `pip install -r requirements.txt`
- `flask db init`
- `flask db migrate -m 'new database' `
- `flask db upgrade`
- `export FLASK_APP=run.py`
- `flask run`
